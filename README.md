## Iniciar aplicación

Install

Ejecutar SCRIPT SQL alojado en la carpeta "DB_SCRIPT"

```
  git clone https://github.com/feblesariel/alkemy-challenge.git
  npm install
    
```
Start

```
    npm test

```

El servidor corre en:

Open [http://localhost:3000](http://localhost:3000)
